  │  address                string   Empty       CPC instance hostname or IP address.         │
  │  port                   string   9999        CPC instance port.                           │
  │  bypass_ping            bool     FALSE       Ping CPC host before reading.                │
  │                                                                                           │
  └───────────────────────────────────────────────────────────────────────────────────────────┘
